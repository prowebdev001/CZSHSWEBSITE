ICT-PROGRAMMING

(Information and Communications Technology)

(Programming)

The ICT-Programming strand is a senior high school program that focuses on developing students' skills and knowledge in the field of information and communications technology (ICT) and computer programming. This strand is designed for students who are interested in pursuing careers in software development, game development, web development, and other related fields. The ICT-Programming strand curriculum includes courses that cover a range of topics such as programming languages, software development methodologies, web development, database management, and computer networks. Students learn how to design, develop, and test software applications, as well as how to analyze and solve technical problems. They also learn how to work with different types of computer systems and platforms.

ICT-CSS

(Information and Communications Technology)

(Computer System Servicing)

ICT-Computer System Servicing (CSS) is a senior high school strand under the Information and Communications Technology (ICT). It is designed to provide students with the knowledge and skills to install, configure, maintain, and repair computer systems and networks. The CSS strand covers topics such as computer hardware and software, network protocols, programming languages, and database management. It also includes training in the use of tools and techniques for troubleshooting and problem-solving. Students in the CSS strand will learn how to assemble and disassemble computer hardware components, install operating systems and application software, set up network infrastructure, configure security protocols, and perform routine maintenance tasks. They will also learn programming concepts, such as variables, data types, loops, and functions.
